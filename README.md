# Descrição
Esse é um repositório que reúne todos os projetos realizados no Curso Introdutório de Arduino implementado na disciplina de Introdução à Computação, componente curricular do 1º período do curso de Ciência da Computação na Universidade Federal de Mato Grosso do Sul.

Esses trabalhos estão todos disponíveis no [Tinkercad](https://www.tinkercad.com/), que é uma plataforma online gratuita da Autodesk para modelagem 3D, simulação de circuitos eletrônicos (incluindo Arduino) e programação visual com blocos, ideal para iniciantes aprenderem design, eletrônica e prototipagem de forma intuitiva e acessível, sem precisar instalar nada.

Os relatórios de todos os projetos estão incluídos nesse repositório por meio de arquivos pdf.

# Projetos
## Desafio 1
Seu time foi contratado para desenvolver um sistema de controle de tráfego eficiente e seguro para uma pequena interseção em uma cidade. A sua tarefa é criar um semáforo inteligente para carros e pedestres utilizando os conhecimentos adquiridos. Este sistema de semáforo deve garantir a segurança dos pedestres ao atravessar a rua e também manter o fluxo de veículos fluindo suavemente.

## Desafio 2
Seu time foi contratado por uma empresa de automação para desenvolver um sistema de controle que será utilizado em um projeto de prototipagem rápida. A sua tarefa é criar um sistema de controle para um servomotor utilizando um potenciômetro, e implementar um feedback sonoro para indicar os limites de rotação.

Neste desafio, você deve controlar um servomotor utilizando um potenciômetro. O objetivo é fazer com que o servomotor realize uma rotação suave conforme a posição do potenciômetro, permitindo um controle preciso da posição do servo. Quando o servomotor atingir a rotação máxima, um buzzer deve emitir um som rápido em uma frequência específica para indicar que o limite foi alcançado. Da mesma forma, quando o servomotor atingir a rotação mínima, o buzzer deve novamente emitir um som rápido.

[Resposta do Desafio 2](https://www.tinkercad.com/things/k4DrxRaJNlI-desafio-2-controle-de-servomotor-utilizando-um-potenciometro-?sharecode=njBiWS52RtkKO_GwgJS8hSmDgF7unvN-XtD1NCSStWE)

## Desafio 4
Seu time foi contratado por uma granja para desenvolver um sistema de iluminação inteligente que ajude a otimizar o bem-estar das aves e economizar energia. O objetivo é criar uma lâmpada que acenda apenas quando certas condições ambientais forem atendidas, garantindo um ambiente adequado para os animais. A lâmpada inteligente que só acenderá sob as seguintes circunstâncias:

- O sensor LDR deve ler um valor abaixo de 400, indicando baixa luminosidade ambiente, garantindo que a iluminação só será ativada quando necessário.
- O sensor de temperatura deve medir uma temperatura abaixo de 25 °C, assegurando que a iluminação adicional seja fornecida apenas quando a temperatura estiver suficientemente baixa.
- O sensor ultrassônico deve detectar a presença de uma pessoa ou animal a uma distância de até 150 cm, garantindo que a luz seja acionada apenas quando houver presença no local.

[Resposta do Desafio 4](https://www.tinkercad.com/things/57Fw52if7AM-desafio-4-luz-inteligente?sharecode=Lckl5uOKSbIyjrgF3tAMpv4Jy0qzF11SUssX6Gt-pNM)

## Desafio 5
Para este desafio, você deve montar um sistema de segurança. Este sistema consiste em verificar senha de 5 dígitos digitada pelo usuário. Para isso, utilize um teclado matricial com um LCD. O usuário tem 3 chances de digitar a senha, caso atinja o limite, deve esperar um tempo para tentar novamente. Também existem os comandos de alterar senha (escrevendo "AAAAA") e deletar dígito (apertando a tecla "B"). A senha que aparece no display deve ser criptografada. Fragmente seu código em funções para melhor desempenho.

Dica de funções da linguagem c++ para manuseio de strings:

- strcmp();
- strlen();
